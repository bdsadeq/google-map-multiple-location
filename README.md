# Google Map Multiple Location (Bangladesh Map)



### View Customization
If you need 100% view (no scroll) then use this
```css
    #map_wrapper {
        height: 100%;
    }
```



### Label Display
If you need to display marker with label, enable this code
```javascript
    label: {
        color: '#299b06',
        fontWeight: 'bold',
        text: markers[i][0],
    }
```
